Langfuse: платформа наблюдаемости для LLM-приложений

Langfuse — это open-source платформа для трассировки, оценки и управления промптами
в LLM-приложениях. Ключевое преимущество — возможность self-hosted развёртывания,
что позволяет хранить все данные локально.

Основные возможности:

1. Трассировка (Tracing)
- Автоматический сбор данных о каждом LLM-вызове
- Иерархическая структура: Trace → Span → Generation → Event
- Поддержка вложенных вызовов для сложных пайплайнов
- Метаданные: user_id, session_id, tags, metadata
- Временные метки для анализа латентности

Способы инструментации:
a) Декоратор @observe() — оборачивает Python-функции, автоматически создаёт
   спаны и вложенность по стеку вызовов
b) OpenAI Wrapper — drop-in замена для from langfuse.openai import OpenAI,
   автоматически трассирует все вызовы OpenAI-совместимых API
c) LangChain CallbackHandler — автоматическая трассировка цепочек и агентов
d) Low-level SDK — ручное создание трейсов, спанов и генераций

2. Prompt Management (управление промптами)
- Централизованное хранение промптов с версионированием
- Метки: production, staging, latest для управления жизненным циклом
- Переменные {{variable}} для динамической подстановки
- Chat-формат и text-формат промптов
- Привязка промптов к генерациям для отслеживания использования
- A/B-тестирование разных версий промптов

3. Evaluation (оценка)
Типы оценок:
- Numeric (числовые): 0.0-1.0 для метрик релевантности, качества
- Categorical (категориальные): "good"/"bad", "safe"/"unsafe"
- Boolean (булевые): True/False для бинарных проверок

Способы оценки:
- Ручная оценка через UI (human-in-the-loop)
- Автоматическая через SDK (score_current_trace, score_current_observation)
- LLM-as-Judge: использование другой модели для оценки

4. Datasets & Experiments (датасеты и эксперименты)
- Создание наборов тестовых данных: input + expected_output
- Запуск экспериментов: прогон приложения по датасету
- Сравнение ранов: side-by-side сопоставление результатов
- Regression testing: проверка что новая версия не хуже предыдущей

5. Analytics (аналитика)
- Дашборд с ключевыми метриками: стоимость, латентность, количество запросов
- Фильтрация по пользователям, моделям, тегам
- Экспорт данных через API для внешней аналитики
- Агрегация скоров для отслеживания качества во времени

Архитектура развёртывания:

Self-hosted (Docker Compose):
- Langfuse Server (Next.js приложение) — порт 3000
- PostgreSQL — хранилище данных
- Redis — кэш и очереди
Все данные остаются на вашем сервере.

Cloud (cloud.langfuse.com):
- EU и US регионы
- Бесплатный тариф для экспериментов
- Managed инфраструктура

Python SDK:
pip install langfuse

Инициализация:
from langfuse import Langfuse
langfuse = Langfuse()  # читает из переменных окружения
# LANGFUSE_PUBLIC_KEY, LANGFUSE_SECRET_KEY, LANGFUSE_HOST

Интеграции: OpenAI, LangChain, LlamaIndex, LiteLLM, Haystack, и другие.
